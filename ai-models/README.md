# AI Models

General-purpose AI/ML models for the EEG–fNIRS project, split by paradigm.

```
ai-models/
├── deep-learning/      # Neural networks (CNNs, RNNs, Transformers, …)
└── machine-learning/   # Classical ML (SVM, RF, XGBoost, …) + feature pipelines
```

> The **EEG → fNIRS synthesis** task lives in its own top-level folder
> (`../cross-modal-synthesis/`) because it is the project's core contribution.
> Use this folder for supporting/auxiliary models such as classification,
> regression, artifact rejection, and baselines.
